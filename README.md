使用 Faster Whisper 和 Google Gemini 进行视频字幕生成与翻译

在 Google Colab 环境下：

1. 使用 Faster Whisper 对视频 / 音频进行语音识别，并生成字幕文件
2. 利用 Google Gemini 对源字幕进行翻译，并生成双语字幕文件

你需要准备一个 Google Gemini API。
